1. File detect_motion.py is used to detect motions on camera. Before run this python script, make sure the python packages cv2 and numpy are already installed on your server. And, remember to set up the saved directory in variable `save_directory`.
2. File upload_photo_local.py is used to upload the captured photos under the specific diretory. Before run this python script, please make sure the python packages requests and base64 are already installed on your server. And remember to set up the photo saving directory in variable `photo_saved_dir` and remtoe uploading url in variable `remote_url`. (Notice: you need to deploy the remote photo uploading service on your server.)
